Projects

iTasteIt.com

This is my primary project. It's a Joomla-based review site for food and beverage products that utilizes a commercial component for most of it's functionality. Be sure to view it on your smartphone as well. The user experience is intentionally very different. It's pairs well with tasting rooms and events.

Bisky's Page

Just a fun little project. I was interested in learning the Google Maps API so I started recording my walks with our dog so I could display them on a map. Sadly, he died last August so I created this as a kind of tribute page for him. I took several pictures of him with my iPhone so I added these as well. Be sure to zoom out to see pics of him in other parts of the country. I'd like to improve on it as time permits and eventually add it to my GitHub profile. Even though we have a new puppy my wife and I miss him a lot... :(

OMDB

Search The Open Movie Database . This was a project for Javascript Foundations course at Galvanize. It utilizes their public API to search the database and display the results. Users can search by title, year and media type.
